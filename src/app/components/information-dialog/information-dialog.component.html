How to create a server ?

See this image, the detail steps are below:

1. Click "Add Icon" button on left side panel

2. Enter redis values (name,host,port,password,database index)

3. Click Add button

 How to connect to an existing server ?

See this image, in fact, there is no explicit connect action

when redis instance icon is "Two arrows", that mean the redis is disconnect status. just click it, it will start auto connect.

 How to disconnect from a server ?

See this image, the detail steps are below:

1. select the redis instance that you want to disconnect.

2. click "Disconnect" button

 How to unregister/delete a redis server?

See this image, the detail steps are below:

1. select the redis instance that you want to disconnect.

2. click "Delete" button

3. click "YES" button
